# 🛒 Laravel E-commerce API

Professional REST API built with Laravel for modern e-commerce applications.

## 🚀 Features
- 🔐 JWT Authentication
- 📦 Product Management
- 🛍️ Shopping Cart
- 💰 Order Processing
- 💳 Payment Integration
- 📊 Admin Dashboard APIs

## 🛠️ Tech Stack
- **Framework:** Laravel 11
- **Database:** MySQL
- **Authentication:** JWT
- **Documentation:** Swagger/OpenAPI
